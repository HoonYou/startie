# Docs Guide

This folder is used to connect team activity to CEO decision-making.

## Folder Structure

- `meetings/`
  - Meeting notes
  - Recommended: agenda, decisions, action items, owners, due dates
- `worklog/`
  - Daily work logs
  - Recommended: what was done, what was checked, blockers, next actions
- `decisions/`
  - Decision records
  - Recommended: decision, background, alternatives, expected impact
- `analysis/`
  - Q1~Q5 analysis briefs and summary documents
  - Recommended: question definition, key metrics, tables used, so what
- `metrics/`
  - Metric definitions and calculation rules
- `templates/`
  - Reusable templates for the folders above

## Operating Principle

- The purpose is not chart production. The purpose is CEO decision support.
- Analysis documents should be organized around Q1~Q5 whenever possible.
- Insight summaries should follow:
  - What
  - Why
  - So What
  - Impact

## Recommended File Naming

- Meeting note: `YYYY-MM-DD_topic.md`
- Worklog: `YYYY-MM-DD.md`
- Decision record: `YYYY-MM-DD_decision_topic.md`
- Analysis brief: `q1_activation.md`, `q2_channel_quality.md`
